## О проекте

**CS2 Hybrid Optimizer** — open-source утилита для повышения плавности работы **Counter-Strike 2**.

Проект объединяет удобство **Python** и производительность нативного **C/WinAPI** кода для очистки системы, обслуживания кэшей, тонкой настройки планировщика ОС и автоматического запуска игры.

Основные цели:

- Улучшение frame time и стабильности фреймтайминга
- Повышение стабильности 0.1% Low FPS
- Снижение микрофризов при компиляции шейдеров
- Очистка накопившегося мусора
- Более плавный игровой процесс

## Возможности

### Кэш и память

| Функция | Что делает | Эффект |
| --- | --- | --- |
| Сброс GPU-кэша | Очищает D3DSCache, NVIDIA DXCache/GLCache и AMD DxCache/OglCache/VkCache | Возвращает плавность после обновлений драйверов или игры |
| Очистка Steam Shader Cache | Удаляет устаревшие данные шейдеров из кэша AppID 730 | Быстрее загружаются карты |
| Сброс RAM Standby List | Вызывает недокументированную функцию `NtSetSystemInformation` для освобождения страниц памяти, удерживаемых в standby-листе | Освобождает память без выгрузки рабочих наборов других запущенных программ |
| Точный системный таймер | Запрашивает максимально доступную точность таймера через `NtSetTimerResolution` (обычно ~0.5мс вместо стандартных ~15.6мс) | Снижает джиттер фреймтайминга; автоматически освобождается при выходе из программы |

### Системные настройки

| Функция | Что делает | Эффект |
| --- | --- | --- |
| Умное распределение ядер | Подсчитывает потоки процессора и делит их на два для использования физических ядер (учитывает SMT/Hyper-Threading) | Уменьшает лаги при компиляции шейдеров |
| Высокая производительность питания | Активирует схему питания "Высокая производительность" | Предотвращает троттлинг CPU во время игры |
| Отключение Xbox GameBar / GameDVR | Отключает ключи реестра `AppCaptureEnabled` и `GameDVR_Enabled` | Убирает накладные расходы от фоновой записи |
| Настройка Win32PrioritySeparation | Переключает планировщик Windows на короткие фиксированные кванты с приоритетом активного окна | Повышает отзывчивость активного окна |
| Твик MMCSS SystemResponsiveness | Устанавливает `SystemResponsiveness` в 0, чтобы фоновые задачи не резервировали CPU во время игры | Больше ресурсов CPU для `cs2.exe` |
| Отключение Fullscreen Optimizations | Отключает DX-оптимизацию эксклюзивного полноэкранного режима именно для `cs2.exe` | Может снизить задержку ввода в полноэкранном режиме |
| Очистка DNS-кэша | Выполняет `ipconfig /flushdns` | Убирает устаревшие DNS-записи перед подключением к матчмейкингу |

### Запуск и удобство

| Функция | Что делает | Эффект |
| --- | --- | --- |
| Автоматическое исключение Defender | Добавляет папку CS2 в исключения Защитника Windows (отображается в консоли и логе) | Уменьшает фоновые проверки во время игры |
| Защита от зависаний | Завершает зависшие процессы `cs2.exe` перед запуском | Исключает конфликты запуска |
| Генератор autoexec.cfg | Создаёт настроенный `autoexec.cfg` (сетевые параметры, интерполяция, fps_max) в папке cfg CS2 | Одинаковые настройки при каждом запуске |
| Идеальный запуск | Генерирует оптимальные параметры запуска и запускает CS2 через Steam | Стабильный старт игры |
| Повышение приоритета после запуска | Через несколько секунд после запуска устанавливает приоритет `cs2.exe` на "Высокий" через PowerShell | Более плавное распределение CPU для процесса игры |

## Генерируемые параметры запуска

Программа автоматически формирует параметры вида:

```bash
-high
-nojoy
-fullscreen
-threads X
+fps_max 0
+exec autoexec
```

Где **X** рассчитывается автоматически на основе количества физических ядер процессора.

## Алгоритм работы

1. Расчёт оптимальных параметров запуска на основе обнаруженных потоков/ядер CPU.
2. Поиск установленной CS2 через реестр Windows и добавление папки установки в исключения Defender.
3. Завершение зависших процессов `cs2.exe` перед стартом.
4. Оценка текущего размера кэша для итогового отчёта.
5. Вызов Си-движка: очистка GPU-кэшей шейдеров, очистка Steam Shader Cache, сброс RAM standby list и запрос точного системного таймера.
6. Очистка DNS-кэша и переключение Windows на схему питания "Высокая производительность".
7. Отключение Xbox GameBar/GameDVR, настройка `Win32PrioritySeparation` и `SystemResponsiveness` для приоритета активного окна.
8. Генерация `autoexec.cfg` и отключение Fullscreen Optimizations для CS2, если установка найдена.
9. Вывод итогового отчёта и, при подтверждении, запуск Counter-Strike 2 через Steam — с повышением приоритета процесса через несколько секунд после запуска.

## Использование

### 1. Скачайте релиз

Скачайте:

```text
CS2_Hybrid_Optimizer.exe
```

из раздела Releases.

### 2. Запустите от имени администратора

```text
Запуск от имени администратора
```

Требуется для очистки кэшей, настройки реестра и конфигурации Defender.

### 3. Выберите язык

```text
1 - English
2 - Русский
```

### 4. Дождитесь завершения оптимизации

Си-движок и Python-слой:

- Очистят кэши
- Сбросят RAM standby list
- Настроят системный планировщик и таймер
- Подготовят операционную систему
- Удалят устаревшие шейдеры

### 5. Запустите игру

В конце появится вопрос:

```text
Запустить Counter-Strike 2 прямо сейчас? (Y/N)
```

| Ввод | Результат |
| --- | --- |
| Y | Запуск CS2 с оптимизациями |
| Enter | Запуск CS2 с оптимизациями |
| N | Выход из программы |

## Безопасность

Проект:

- Не внедряется в процесс CS2
- Не изменяет память игры
- Не взаимодействует с VAC-модулями
- Не модифицирует игровые файлы
- Работает исключительно с кэшами Windows/Steam и системными настройками уровня ОС (реестр, схема питания, планировщик)
