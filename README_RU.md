## 📖 О проекте

**CS2 Hybrid Optimizer** — это open-source утилита для повышения плавности работы **Counter-Strike 2**.

Проект объединяет удобство **Python** и производительность нативного **C/WinAPI** кода для очистки системы, обслуживания кэшей, оптимизации запуска игры и автоматической подготовки Windows перед матчем.

Основные цели:

* 🎯 Улучшение Frame Time
* 📈 Повышение стабильности 0.1% Low FPS
* ⚡ Снижение микрофризов при компиляции шейдеров
* 🧹 Очистка накопившегося мусора
* 🎮 Более плавный игровой процесс

## ✨ Возможности

| Функция                       | Что делает                                                                         | Эффект                                 |
| ----------------------------- | ---------------------------------------------------------------------------------- | -------------------------------------- |
| 🧠 Умное распределение ядер   | Подсчитывает потоки процессора и делит их на два для использования физических ядер | Уменьшает лаги при компиляции шейдеров |
| 🎮 Сброс GPU-кэша             | Очищает DXCache и GLCache NVIDIA, AMD и DirectX                                    | Возвращает плавность после обновлений  |
| 🔥 Очистка Steam Shader Cache | Удаляет мусор из shadercache AppID 730                                             | Быстрее загружаются карты              |
| 🧹 Оптимизация памяти         | Вызывает `SetProcessWorkingSetSize()` для очистки рабочей области процесса         | Освобождает память                     |
| 🛡 Исключение для Defender    | Автоматически добавляет папку CS2 в исключения Защитника Windows                   | Уменьшает фоновые проверки             |
| 🚨 Защита от зависаний        | Завершает зависшие процессы `cs2.exe` перед запуском                               | Исключает конфликты запуска            |
| 🚀 Идеальный запуск           | Генерирует оптимальные параметры запуска и запускает игру через Steam              | Стабильный старт игры                  |

## ⚡ Генерируемые параметры запуска

Программа автоматически формирует параметры вида:

```bash
-high
-nojoy
-threads X
+fps_max 0
```

Где **X** рассчитывается автоматически на основе количества физических ядер процессора.

---

## 🔬 Алгоритм работы

1. Поиск установленной CS2 через реестр Windows.
2. Завершение зависших процессов игры.
3. Очистка GPU-кэшей.
4. Очистка Steam Shader Cache.
5. Оптимизация памяти.
6. Настройка исключений Defender.
7. Расчёт оптимального количества потоков.
8. Генерация параметров запуска.
9. При необходимости запуск игры через Steam.
