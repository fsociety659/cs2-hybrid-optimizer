## ⚙️ Как это работает (Архитектура)

| Компонент | Роль | Технологии |
|-----------|------|-------------|
| 🐍 **Python (мозг)** | Умный интерфейс: определяет потоки CPU, пути к CS2, управляет логикой | `psutil`, `pywin32`, `PyInstaller` |
| ⚡ **C-мотор (cleaner.dll)** | Бесшумный чистящий двигатель: работает напрямую с WinAPI, ядром Windows, без вызовов system() | `WinAPI`, `DirectX`, `kernel32` |

> ✅ **Python** — удобство и контроль.  
> 🔥 **C** — скорость и тишина (никаких консолей, только чистое ядро).

---

## 🚀 Основные фичи

| Фича | Что делает | Эффект |
|------|------------|--------|
| 🧠 **Умное распределение CPU** | Автоматически делит логические потоки на 2 (Hyper-Threading / SMT). CS2 работает строго на физических ядрах. | Убирает **shader stuttering** |
| 🗑️ **Сброс кэша графики** | Чистит папки `DXCache`, `GLCache` (NVIDIA / AMD / DirectX) | Возвращает плавность после обновлений |
| 🧹 **Очистка шейдеров Steam** | Удаляет мусор шейдеров для CS2 (AppID: 730) | Ускоряет загрузку карт |
| 💾 **Очистка ОЗУ (Standby List)** | Освобождает «забитую» оперативную память (как RAMMap) | Больше свободной RAM для игры |
| 🛡️ **Обход Защитника Windows** | Добавляет папку CS2 в исключения Defender'а | Defender не грузит CPU во время катки |
| 🔒 **Защита от блокировок** | Принудительно закрывает зависшие `cs2.exe` | Никаких «игра уже запущена» |
| ⚡ **Идеальный автозапуск** | Формирует параметры запуска (`-high -nojoy -threads X +fps_max 0`) и открывает игру через Steam | Максимальный FPS без лагов |

## 📖 Инструкция по использованию

1. 📥 **Скачай** файл `CS2_Hybrid_Optimizer.exe` из раздела [Releases](../../releases)
2. 🖱️ **Запусти** от имени администратора
3. 🌐 **Выбери язык:** `1` — English, `2` — Русский
4. 🎮 **Жди 5 секунд** — софт всё сделает сам
5. ✅ **Играй плавно** — без stutter'ов и фризов

> 💡 **Важно:** Никакого Python или C ставить не нужно. Всё уже внутри одного `.exe`-файла.
